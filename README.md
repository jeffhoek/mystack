# mystack
[Docker](https://www.docker.com/) (Podman) [Compose](https://podman-desktop.io/docs/compose) based stack for training, and generative AI development, and solution prototypes.

## Featured Services

- [MongoDB Atlas](https://www.mongodb.com/products/platform/atlas-database)

- [Postgres (with pgvector)](https://github.com/pgvector/pgvector)

- [OpenSearch (with OpenSearch Dashboards)](https://opensearch.org/)

- [Pinecone local](https://docs.pinecone.io/guides/operations/local-development)
> NOTE: _As of 2026-01-15 pinecone-local still only supports API version 2025-01 which requires SDK 6.*_. See: https://community.pinecone.io/t/pineconeapiexception-422-reason-unprocessable-entity/8196.
